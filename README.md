# BattabotWebUI

## About

This is a <b> Work In Progress</b> WebUI for [BattaBot](https://github.com/FmartinP99/BattaBot/) with Websocket communication.
The program is written in React with NextJs using React Redux and Tailwind.
The goal is to create a Discord-like interface for [BattaBot](https://github.com/FmartinP99/BattaBot/), allowing full control of the bot.

## Features (more is coming)

- Real-time status update for each member.
- Real-time text messaging.
- Image/video preview if a text message contains URLs.
- Support for Channel/User mention behavior.
- Set reminders for each member.
- View existing reminders.
- Set roles for each member.
- Control the music playlist in each server separately (with synchronized playlist and progress bar).
